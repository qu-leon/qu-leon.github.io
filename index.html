TruePath

Python program that helps to track the processing of specific wafers in the lot. Certain wafer may be split out for different processing experiments, and this program helps to identify which wafers are part of which experiments.

After implementation, the script significantly improved tracking efficiency as it helped show full flow of a lot and the path of each wafer. It features functionality to show a single lot flow or compare two lots. Files can be generated to Excel or HTML file.
